# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, professional, and collaborative environment for everyone participating in this project.

## Expected Behavior

Participants are encouraged to:

* Be respectful and constructive.
* Focus on technical discussions.
* Accept feedback professionally.
* Share knowledge openly.
* Help improve documentation and code quality.

## Unacceptable Behavior

The following behaviors are not tolerated:

* Harassment or discrimination.
* Personal attacks.
* Deliberate misinformation.
* Publishing confidential information.
* Abusive language.

## Enforcement

Project maintainers reserve the right to remove contributions, comments, issues, or pull requests that violate this Code of Conduct.

## Scope

This policy applies to:

* GitHub Issues
* Pull Requests
* Discussions
* Project communications

By participating in this project, you agree to follow these guidelines.
