# assets\

Repo-branding SVGs used by the root `README.md` (not shipped into any built image -
see `Branding/` for that).

| File | Used for |
|---|---|
| `logo-horizontal-light.svg` | README header logo, light theme (`prefers-color-scheme: light`) |
| `logo-horizontal-dark.svg` | README header logo, dark theme (`prefers-color-scheme: dark`) |
| `logo-square.svg` | Square variant, for contexts that need a 1:1 aspect ratio (e.g. social preview) |

Vector source, safe to re-export or re-theme without touching the README markup itself.
